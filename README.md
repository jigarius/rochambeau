## Installation

    gem install rochambeau

## Usage

    rochambeau # Play rock-paper-scissors classic.
    rochambeau --advanced # Includes Lizard and Spock.
    rochambeau --players 2 # A two-player game.
    rochambeau help # In case you get lost.
